消息窗口使用简介

下面我们来讲讲如何方便快捷的使用jshow for bootstrap的消息窗口。

什么是消息窗口?

消息窗口有点类似alert窗口，进行一些简单的消息提示，但是它不会影响你的正常操作。 只要你引入了相关资源，消息窗口的使用非常简单，就这么简单一句javascript代码:
message_box.show('这是一个消息窗口！');

一、jshow for bootsrap的窗口是如何创建的呢？

消息窗口的使用很简单，你在任何地方调用message_box.show('消息','info|warn|success|error')！ 注意 上面的 "|"的意思是或者，不是全部录入哈~！ 

 //以下代码用来生成各种级别的提示消息
	//提示消息
	<a href="#nogo"  class="btn btn-info" onclick="message_box.info('<h4>信息</h4>只是一个提示消息！');">信息提示</a> 

	//警告消息 默认
	<a href="#nogo"  class="btn btn-warning" onclick="message_box.warning('<h4>警告</h4>只是一个提示消息！');">警告提示 默认</a> 

	//成功消息
	<a href="#nogo"  class="btn btn-success" onclick="message_box.success('<h4>成功</h4>只是一个提示消息！');">成功提示</a> 

	//失败或者错误
	<a href="#nogo"  class="btn btn-danger" onclick="message_box.error('<h4>失败</h4>只是一个提示消息！');">失败或错误提示</a> 
效果如下： 信息提示 警告提示 默认 成功提示 失败或错误提示

二、参数说明

  1. 第一个参数： 消息内容，消息可以为HTML片段，
  2. 第二个参数： 消息类型，info, warn, success,error 4个级别的消息。
  3. 调用示例 ： message_box.show('消息','info');
