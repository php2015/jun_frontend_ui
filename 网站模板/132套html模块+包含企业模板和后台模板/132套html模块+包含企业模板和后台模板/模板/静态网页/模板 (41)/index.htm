This template has been fully tested to be displayed correctly at least with FireFox v1.5 and Internet Explorer v6.0. A minimum of 800x600 pixels of screen resolution is required, but we suggest higher resolutions for your viewing pleasure.

The idea behind summerBreeze...

...is summer time! ;)

summerBreeze is a little project established few months back, when it was just an idea. That would be a CSS driven and XHML 1.1 valid template featuring a special tabbed main navigation and wise use of the white spaces. While summer time was approaching, I got myself to work on some basic code for it, but still in primal beta versions. My free time was really limited, so the completion of the template seemed always far. And then, there was a university assignment that gave me the appropriate motivation to finally complete this template.

All and all, this is a really hard worked template and I feel proud to finally be able to give something I think to be valuable to the web community. The CSS file is widely (and/or wildly) commented, so it will be extremely easy for fellow webmasters to understand it and modify it to their needs.

I hope the visual result feels fresh and reminds of the sea, the mild atmosphere and stuff... Anyway, summer has come and I wish you all to enjoy it as much as you can. Happy vacations!

CSS Limitations

I have another version of this template with one javascript trick that solves a problem which CSS is unable to fix by itself yet. This will be the 3-Column 100% height adjustment according to the longest column (based on how long the content goes). However, javascript is against the rules of submitting a template here, so if anyone's interested, feel free to contact me for details. In the template you are viewing now, I have used a background image to smooth the ending of the left and the right column accordingly.
