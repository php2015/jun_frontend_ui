Welcome !

Hello, this is my first webdesign made for OSWD. I called it blue sky because of the colors I used. It is 100% compliant with XHTML 1.0 Strict and does not contain any tables. I also tried to use relative metrics everywhere, so the layout should be very flexible (scaling, resizing, font sizes, etc.). Further, I also included three different stylesheets (print layout, handheld, larger fonts). I would be very happy about some feedback! - Jonas John

The source code (HTML and CSS) should be tidy and easy to to customize. I tested the design successfully in Mozilla Firefox 1.5, Opera 9.0 and Internet Explorer 6.0. Some parts of the CSS code and the XHTML structure were inspired by the layouts of haran - thanks for the great work!


Feel the fresh air

Feel free to use this template for any purpose you wish (personal, commercial, ...). If you publish a website that is based on blue:sky, please email me your site's URI - that would be very nice (because I'm very curious)!
