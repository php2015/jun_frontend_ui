Welcome to 4o3commerce

Hello and thank you for taking the time for viewing my design!

This is not only my first design for oswd / openwebdesign but also my very first attempt at creating a table-less design from scratch.

This design is created to be used by a shopping cart. The layout is inspired by the classic e-commerce layout used in such applications as zen cart and os commerce.

The design is open source so feel free to use it as it is or modify it to suit your use. I don't require any credit for it, but I would appreciate an e-mail to see how it is being used.

The design validates as xhtml 1.0 strict, the CSS does no due to the opacity hacks which work in IE and mozilla, but not Opera as far as I am aware. Thanks to Infinity for the brainstorming session.
