select插件 bootstrap-lyear-select

该插件是基于bootstrap的一个简单的select下拉框插件，只支持单个选项(不支持多选项和分组)、搜素。使用时，记得引入页面style内的样式。

之所以做这个插件，只是为了配合后面的联动插件一起使用，当然你也可以不用管，直接用原生的。

简单使用
默认有禁用值和选中值
可搜索的
设置宽度

200px宽度

50%宽度

布局宽度
$('.simpleSelect').each(function() {
    var $this = $(this);
    $this.lyearSelect({
        width: $this.data('width') ? $this.data('width') : '',                          // 宽度width，示例：75px、50%
        placeholder: $this.data('placeholder') ? $this.data('placeholder') : '请选择',   // 空值项提示语，默认：请选择
        search: $this.data('search') ? $this.data('search') : false,                    // 是否允许搜索
    });
});
