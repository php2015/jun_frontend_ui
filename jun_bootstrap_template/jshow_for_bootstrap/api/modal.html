bootstrap原生的弹窗有什么缺点?

  1. 不能弹窗多层窗口，也很难对弹出窗进行管理。
  2. 生成ajax窗口复杂，不方便，而且兼容性不好？个人认为~~
  3. modalmanager虽然可以弹多层窗口，但是使用也较麻烦，而且不能直接关闭顶层窗口，窗口关闭后无法回收dom资源!
所以，jshow for bootstrap 诞生了，它解决了以上所有问题，当然它还有更多美好的功能！

一、jshow for bootsrap的窗口是如何创建的呢？

本系统解决了以上所有难题，同时也兼容之前所有的弹窗的方法，您原来的弹窗控件依然可以继续使用！ 


//以下代码用来生成一个按钮
<a href="#nogo" id="openDialog" class="btn  btn-info">很简单的JS点击弹窗</a> 

//以下脚本用来将弹窗事件绑定到按钮上

	$("#openDialog").click(function(){
		gDialog.fCreate({title:'这是一个弹窗窗口',url:'myDialog.html',width:700}).show();	
	});

//在任何时候通过以下代码，可以关闭顶层弹窗
	gDialog.fClose();
警告! 由于本地文件访问权限原因，非IE或者firefox浏览器本弹窗可能会内容为空，建议IE或者firefox浏览，谢谢！
效果如下： 很简单的JS点击弹窗

二、参数说明

  1. title： 标题，可为空
  2. url ： 请求资源地址，也就是你窗口内容的地址
  3. content ：如果非url请求，可以通过content直接设定dialog中的值，当url为空的时候才起作用
  4. width： 宽度，也就是窗口的宽度
  5. reuse： 重复使用，如果该参数为true，该窗口关闭后，资源不会被回收
  6. dragConfig：传递jquery ui draggable的 拖动参数配置，非必输项，如果需要指定拖动的handle节点，可以通过：dragConfig:{handle:'选择器'}，来实现该功能！

代码如下： 
	<!-- Button to trigger modal -->
	<a href="#myModal" role="button" class="btn" data-toggle="modal">查看演示案例</a>
	 
	<!-- Modal -->
	<div id="myModal" class="modal hide fade" tabindex="-1" role="dialog" aria-labelledby="myModalLabel" aria-hidden="true">
	  <div class="modal-header">
		<button type="button" class="close" data-dismiss="modal" aria-hidden="true">×</button>
		<h3 id="myModalLabel">Modal header</h3>
	  </div>
	  <div class="modal-body">
		<p>One fine body…</p>
	  </div>
	  <div class="modal-footer">
		<button class="btn" data-dismiss="modal" aria-hidden="true">关闭</button>
		<button class="btn btn-primary" onclick="gDialog.fClose();">js关闭任意弹窗！</button>
	  </div>
	</div>
2.原生窗口支持拖动
在原生窗口中，在任意节点增加 class: drag-handle，即可实现原生窗口的拖动功能， 且可拖动区域在加了该class 的节点上，如下所示。
唯一的缺憾是，拖动后，第二次展现出来的效果有点问题，但不影响使用！

查看演示案例

头部可以拖动

内容区域不能拖动，因为拖动的drag-handle样式在头部区域定义!…

	<!-- Button to trigger modal -->
	<a href="#myModalDrag" role="button" class="btn" data-toggle="modal">查看演示案例</a>
	 
	<!-- Modal -->
	<div id="#myModalDrag" class="modal hide fade" tabindex="-1" role="dialog" aria-labelledby="myModalLabel" aria-hidden="true">
	  <div class="modal-header  drag-handle ">
		<button type="button" class="close" data-dismiss="modal" aria-hidden="true">×</button>
		<h3 id="myModalLabel">Modal header</h3>
	  </div>
	  <div class="modal-body">
		<p>One fine body…</p>
	  </div>
	  <div class="modal-footer">
		<button class="btn" data-dismiss="modal" aria-hidden="true">关闭</button>
