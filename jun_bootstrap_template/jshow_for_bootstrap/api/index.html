jshow for bootstrap是由david zhang整理改造的一款可用于搭建一个功能完善的站点的bootstrap插件集合。本系统基于Bootstrap进行构造，感谢Bootstrap这一优秀的开源软件！

本集合包含了modalManager开源插件，对modalManager进行了一定的代码改造，使其更方便使用，也更利于垃圾回收。同时，对ajax请求及导航更新以及表单验证等增加了很多神奇的功能。

1.为什么创建它？

我想用bootstrap搭建自己的站点，本人找了很多插件，找了很多资源，它们都无法满足本人的需求。所以，为什么不去融合它，改造它，让它变得更加易用，更加方便。所以，我就这么作做了！哈哈.........

当然，也许它还缺少一些插件？ 是的很多插件，日期插件，拖拽插件，树状插件，富文本插件... 因为我目前不需要他们，所以没有引入进来，热心的朋友如何添加进来之后，可以联系我增加到源码之中。

2.我能用它做什么？

  1. 快速整合建站资源，搭建出方便使用的后台管理代码。
  2. 方便易用的ajax弹出窗口，封装后更简单，可以多层窗口，窗口关闭资源即被回收（只会复制粘贴的菜鸟是无法理解的）。
  3. 同时也支持bootstrap原生的所有弹窗方式。
  4. 更方便的消息提示窗口。
  5. 更简单易用，功能强大的表单验证器。

以上这一切都够了，它们已经足够吸引你了，不是吗？

3.使用它，我需要哪些基础知识？

什么？不懂javascript，jquery,不懂bootstrap，不懂HTML，我劝你还是回去好好补习基础知识吧。只有这些东东你都很熟悉之后，才有驾驭它的本事........不然的话不要惹我说脏话，滚粗...

4.如何使用它？

你需要将misc目录下的除了other目录下的所有文件引入工程，是的，全部！
然后你就可以使用了，类似下面的用法,但是你需要修改一下资源路径，对html驾轻就熟的你应该不成问题： 

		
<link href="../misc/css/bootstrap.min.css" rel="stylesheet" type="text/css" />
<link href="../misc/css/bootstrap-modal.css" rel="stylesheet" type="text/css" />
<link href="../misc/css/joywindow.bootstrap.css" rel="stylesheet" type="text/css" />

<script type="text/javascript" src="../misc/js/jquery-1.7.2.js"></script>
<script type="text/javascript" src="../misc/js/bootstrap.js"></script> 
<script type="text/javascript" src="../misc/js/bootstrap-modal.js"></script>
<script type="text/javascript" src="../misc/js/bootstrap-modalmanager.js"></script> 
<script type="text/javascript" src="../misc/js/modal.manager.plugin1.0.js"></script>
<script type="text/javascript" src="../misc/js/jshow.utils.js"></script>



<!-- you can delete follow resource 如果你不需要拖拽，以下文件可删除 -->
