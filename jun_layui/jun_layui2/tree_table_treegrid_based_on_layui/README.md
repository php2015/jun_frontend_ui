20180919

        1、定义srot排序方式，不再支持前端排序，排序只能是服务器端，点击可排序列，会触发查询语句重新请求后，并将排序字段发到后端
        格式：sort:[{"field":"id","sort":"desc"},{"field":"pId","sort":"desc"}]
