# 代码生成器

#### 项目介绍
Java代码生成器，通过构造自定义模板，可在线生成entity、xml、dao、service、html、js、sql代码，节省编写重复代码的时间，把更多的时间用在思考上

#### 软件架构
- 核心框架：Spring Boot
- 持久层框架：MyBatis
- 项目管理：Maven

#### 安装教程

1. 通过git下载源码
2. 修改application.yml，更新MySQL账号和密码、数据库名称
4. IDEA、Eclipse运行MintLeafApplication.java，则可启动项目
3. 项目访问路径：http://localhost:8060
